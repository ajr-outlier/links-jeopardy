# Links Jeopardy

A Remix application for the Links Jeopardy game.

## Getting Started

Install dependencies:

```bash
npm install
# or
bun install
```

Run the development server:

```bash
npm run dev
# or
bun dev
```

Open [http://localhost:5173](http://localhost:5173) with your browser.

## Build

```bash
npm run build
npm start
```

## Type Checking

```bash
npm run typecheck
```

## Environment Variables

Create a `.env` file for your environment variables if needed.
